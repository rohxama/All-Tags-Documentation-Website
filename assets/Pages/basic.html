Basic Tags

First of all, basic tags are essential for every website.Here are some tags shown below:

  • Doctype tag defines the document type.
  • Html tag defines the html document type.
  • Head tag contains title, meta tags and imports.
  • Title tag defines the title of doucment.
  • Meta tag provide information to browser. It consists of some attributes.Charset attribute defines the encoding type.
  • Body tag contains media, tables and forms etc.
  • Heading tag defines the heading of site.
  • Paragragh tag inserts paragraph.
  • Br tag inserts break line in paragraph
  • Hr tag inserts horizontal line.
