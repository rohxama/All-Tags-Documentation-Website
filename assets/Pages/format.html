Formatting Tags

Formatting tags is used to format the content.Here are some tags shown below:

  • Abbr tag defines abbrevation. It consists of title attribute which give advisory information.
  • Address tag defines the address of author. It consists of some child elements.Anchor tag defines a hyperlink.
  • B tag used to bold the text.
  • Bdi tag defines the text in other languages.
  • Bdo tag is used to change the direction of text. It consists of direction attribute which change direction from right to left.
  • Blockqoute tag defines the section that is copied from other section It consists of cite attribute defines the source.
  • Cite tag defines the title of some work.
  • Code tag defines a piece of computer code.
  • Del tag defines the deleted text. It has a child element.Ins tag insert new text.
  • Dfn tag defines the defination in content.
  • Em tag defines the emphasized text.
  • I tag is used to bold the text.
  • Mark tag is used to mark a text.
  • Pre tag defines the prefromatted text.
  • Q tag defines a short question.
  • Small tag defines the smaller text.
  • Strong tag defines important text.
  • Sub tag defines the subscripted text.
  • Sup tag defines the superscript text.
  • U tag inserts underline umder text.
