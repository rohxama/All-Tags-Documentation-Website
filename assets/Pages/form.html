Form Tags

Forms and input tags are used to create forms.

  • Form tag is used to create form. It has many child elements like input, textarea, label, output etc. It consists of many attributes like charset, action, method, target etc.
  • Input tag is used for input control. It has same attributes of form.
  • Label tag defines label in form.
  • Output tag defines the output of calculation.
