Images Tags

Images tag are used to insert and modify images.

  • Img tag inserts image. It contains src attribute which define a source.
  • Figcaption tag is used to defines figure caption.
  • Figure tag contains content like a container.
  • Picture tag contains pictures like a container.

Audio and Video Tags

Audio and video tag are used insert and modify audios and videos.

  • Audio tag is used to insert audio. It contain src attribute which define a source and type attribute which define type of media.
  • Source tag defines multiple media resources.

Tables tags

Table tags are used to insert and modify tables.

  • Table tag is used to inserts table.
  • Caption tag defines a table caption.
  • Th tag defines table header.
  • Tr tag defines table rows.
  • Td tag defines table details.
  • Thead tag defines header content.
  • Tbody tag defines body content.
  • Tfoot tag defines footer content.

Styles and Semantics Tags

These tags inserts styling.Here are some tags given below:

  • Style tag defines style of doucment.
  • Div tag defines a section in document.
  • Span tag defines a header for a section.
  • Header tag defines header for a doucment.
  • Footer tag defines footer of a document.
  • Mian tag defines the main content of a doucment.
  • Section tag defines a section in a document.
  • Article tag defines article.
  • Aside tag defines a side bar in document.
